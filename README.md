# 🍳 Recipe Finder Web Application

A user-friendly web application that helps users discover a wide range of recipes by searching based on ingredients, cuisine type, or dietary restrictions.

## 🚀 Features

- Search for recipes by entering ingredients, selecting cuisine type, or specifying dietary restrictions.
- Displays detailed recipe information including ingredients, preparation steps, and nutrition facts.
- Allows users to save favorite recipes for easy access.
- Responsive design works seamlessly on desktop and mobile devices.
- Intuitive and clean user interface powered by Bootstrap.

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript   
- Spoonacular API  
- Bootstrap

## ⚡ How It Works

1. User enters search criteria (ingredients, cuisine, dietary restrictions).
2. App makes API requests to Spoonacular API to fetch matching recipes.
3. Displays a list of recipes with key information (title, image, brief description).
4. Users can click a recipe to view full details: ingredients, steps, and nutrition data.
5. Users can save favorite recipes for later reference.
